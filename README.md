课程大作业 
一、 本项是一个使用 C++ 语言和 Cocos2d-x 3.17 引擎开发的跨平台休闲游戏 Visual Studio 2022 / Cocos2d-x 3.17.2 
二、 数据结构设计 
玩家实体：使用 cocos2d::Sprite 对象，通过自定义变量 _velocity 记录实时垂直速度。 
对象池管理：使用 std::vectorcocos2d::Node* 容器存储屏幕上的障碍物，便于统一遍历进行碰撞检测和内存释放 
三、 核心技术实现点 
1 物理模拟（手写非引擎版本） 为了展示对物理公式的理解，代码没有使用 Box2D 物理引擎，而是手动实现了受力逻辑： 重力应用：在每帧更新函数中，使速度按固定比例递减。 
跳跃脉冲：当接收到点击事件时，瞬间赋予玩家一个向上的初速度。 
2 坐标系转换与碰撞判定 这是本项目最复杂的部分。障碍物被放置在移动的 Node 容器中，因此它们的本地坐标是静态的。 
技术细节：通过 obs->convertToWorldSpace(child->getPosition()) 将障碍物的内部坐标实时映射到屏幕坐标系，从而与玩家的 BoundingBox 进行重叠检测。
